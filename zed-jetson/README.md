# zed-ros2-wrapper on Jetson

For running Stereolabs' `zed-ros2-wrapper` on the Nvidia Jetson Orin Nano, on ROS2 Jazzy (which requires Ubuntu 24.04), in a Docker container, on JetPack 6.2.2 (which is based on Ubuntu 22.04).

## Running the container on the Jetson

```
git clone https://github.com/oaklandrobtoics/ros_ora26
cd ros_ora26/zed-jetson
docker compose up
```
